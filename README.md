# Advancing Multi-Tool Usage with Chain-of-Abstraction Reasoning

This repository captures research work around improving LLM factuality and reasoning by combining tool use, retrieval, and chain-of-abstraction-style prompting. It includes notebooks, evaluation artifacts, benchmark subsets, and experiment outputs tied to a Meta-published paper workflow.

## Research Focus

- Multi-tool reasoning for large language models
- Retrieval-augmented grounding from sources like Wikipedia
- Benchmarking across commonsense, math, and multi-hop QA tasks
- Comparing baseline and tool-augmented generation quality

## Repository Structure

- `Multitool1.ipynb`, `Multitool3.ipynb`: core reasoning experiments
- `hotpot_qa.ipynb`: multi-hop question answering work
- `common_sense_qa.ipynb` and `common_sense_chain.ipynb`: commonsense evaluation
- `math.ipynb`: calculator and reasoning experiments
- `Wiki_tool/`: Wikipedia-backed dataset generation and analysis
- `multitool_commons/`: commonsense tool-use outputs and validation files
- `testing data/`: held-out test artifacts

## Skills Demonstrated

- NLP research experimentation
- Tool-augmented prompting
- Retrieval-augmented generation
- Benchmark design and evaluation
- Notebook-based reproducible analysis
- LLM hallucination reduction strategies

## Notes

This repository is best understood as a research artifact and experiment archive rather than a production application.
